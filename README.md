# MyDataApp - UTS Pemrograman Android

Aplikasi Android "MyDataApp" yang dibangun untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Pemrograman Android.

## Fitur Utama
- **Login Screen**: Sistem keamanan sederhana menggunakan SharedPreferences.
- **Credential Validation**: Verifikasi hardcode untuk akses masuk.
- **Dashboard**: Menampilkan pesan selamat datang personal.
- **Data Entry**: Form input data (NIM, Nama, Prodi, Kelas, Alamat, E-Mail).
- **Dynamic List**: Menampilkan data yang diinput menggunakan ListView.
- **Session Management**: Fitur Logout untuk menghapus sesi pengguna.
